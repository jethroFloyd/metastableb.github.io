Multiple Instance Learning for Sequential Data Classification on Resource Constrained Devices
We explore the problem of sequential data (time series) classification on resource constrained devices, specifically, RNN based classification approaches. Such approaches are expensive and prohibitive in the resource constrained setting. With a novel Multiple Instance Learning (MIL) like formulation, that exploits temporal nature of the data, we are able to reduce compute and memory requirements of RNNs by upto 8x-72x which enables us to push such RNN based classifiers on to resource constrained devices. (Accepted to NIPS '18).

GesturePod: Programmable Gesture Recognition for Augmenting Assistive Devices
We look at the problem of on-device gesture detection in the resource constrained setting with a focus on accessibility. Powered with the EdgeML algorithm ProtoNN, we are able to bring machine learning based gesture detection to the arduino class devices. The device we have created, GesturePod, is reliable and robust to real world situations. The ML based approach as opposed to a rule based approach gives as extensibility in the sense that users can add new gestures to the system with very low effort - programmability. (In submission, CHI '19).

WakeWord: Keyword Spotting on a Pi0
We have set for ourselves the ambitious target of getting keyword spotting, or wake-word detection onto an Arduino class device without employing speech specific methods. Exploiting various novel ideas from EdgeML we have made wake-word detection available on Raspberry Pi3 and have recently been able to make the same available on a Raspberry Pi0. Arduino class devices are about 10x speedup away and we believe we should be able to achieve this in the coming few months.



Nagging Naagin: Q-Learning for Snake
Exploring fundamental ideas in reinforcement learning by teaching an agent to play the classic game - Snake. We create a modular and configurable snake simulation with two agents - the snake and an adversarial agent who controls where the food is placed. This multi bandit formulation allows us to explore a number of algorithms with various levels of sophistication - reflex agents, min-max tress, expectimax trees, Q-learning, approximate Q-learning and so forth.

TalkBot: Collaborative Target Tracking for Affordable and Portable Videography
An attempt at automating in-class-room videography at low costs. Multiple Raspberry Pi 3 based recording devices, each powered by a cascaded person of interest tracking stack (MOG + Particle filter + Viola Jones and DNN face recognizers), tracks the speaker and audience from various camera angles and records and broadcasts the talk in real time. All decisions on selecting camera angles, activating appropriate audio devices, identifying the current active speaker and so forth is arrived at through a distributed consensus. The prototype developed was the runner up at Grand Challenge, International Symposium on Embedded Computing and System Design, 2016.

Universal IoT Gateway with Disaster Resilient Communication Pathways
Developed a Universal IoT Gateway - a gateway that can command and communicate with all and any IoT devices, regardless of the communication protocol the specific device supports, be it BLE, Bluetooth, WiFi or ZigBee. This cross-protocol communication is made possible by an ontology based kernel that understands device specific properties and communication atoms. The ability to communicate with multiple protocols allowed the device to also double as a disaster resilient communication pathway - a mesh network at the MAC layer.

Single Cycle RISC-V Micro Architecture Processor and its FPGA Prototype
Developed a RISC-V based single cycle micro architecture processor optimized for low-cost embedded devices. The work also included a compilation stack complete with a fully functional assembler and simulator.
