Multiple Instance Learning for Sequential Data Classification on Resource Constrained Devices
We explore the problem of sequential data (time series) classification on resource constrained devices, specifically, RNN based classification approaches. Such approaches are expensive and prohibitive in the resource constrained setting. With a novel Multiple Instance Learning (MIL) like formulation, that exploits temporal nature of the data, we are able to reduce compute and memory requirements of RNNs by upto 8x-72x which enables us to push such RNN based classifiers on to resource constrained devices.

GesturePod: Programmable Gesture Recognition for Augmenting Assistive Devices
We look at the problem of on-device gesture detection in the resource constrained setting with a focus on accessibility. Powered with the EdgeML algorithm ProtoNN, we are able to bring machine learning based gesture detection to the arduino class devices. The device we have created, GesturePod, is reliable and robust to real world situations. The ML based approach as opposed to a rule based approach gives as extensibility in the sense that users can add new gestures to the system with very low effort.

WakeWord: Keyword Spotting on a Pi0
We have set for ourselves the ambitious target of getting keyword spotting, or wake-word detection onto an Arduino class device without employing speech specific methods. Exploiting various novel ideas from EdgeML we have made wake-word detection available on Raspberry Pi3 and have recently been able to make the same available on a Raspberry Pi0. Arduino class devices are about 10x speedup away and we believe we should be able to achieve this in the coming few months.
